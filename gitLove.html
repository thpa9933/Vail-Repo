Git Lovin
gitLovin'
  • git init --> create empty repo
  • git status --> displays current state
  • git log --> shows commits
  • git add . --> include updates
  • git commit -m " " --> commit updates
  • git push --> upload updates
  • git branch --> check current branch
  • git branch newbranch --> create new branch
  • git checkout branch --> change branch
  • git merge newbranch --> merge branches
